## 如何使用此範本？

1. **Frontmatter (前置區)**:
   - `heroImage`: 文章的主視覺圖。路徑必須以 `/images/` 開頭。
   - `status`: 設定為 `published` 才會正式發布，`draft` 則僅供預覽。
2. **目錄**:
   - 使用 `[顯示文字](#標題名稱)` 建立內部錨點。注意標題名稱要與下方一致。
3. **圖片上傳**:
   - 透過「文章草稿工具」上傳後，系統會告知你圖片路徑（通常是 `/images/YYMM/檔名.jpg`）。
   - 請將該路徑填入 Markdown 語法中。
4. **表格**:
   - 確保標題行與內容行之間有 `| --- |`。
   - 本站 CSS 已處理自動圓角與斑馬紋背景。
